Chapitre 1 — Le Système Nerveux

Cerveau, moelle épinière, nerfs, réflexes, mouvement

📖Résumé simple▼
😊 C'est quoi ?

Le système nerveux, c'est le cerveau + la moelle épinière + les nerfs. Il commande tout le corps. Le message qui circule s'appelle l'influx nerveux.

🔑 Les mots importants
Neurone = cellule du système nerveux Centripète = message qui VA vers le cerveau (sensitif) Centrifuge = message qui SORT vers le muscle (moteur) Réflexe = mouvement automatique très rapide Synapse = contact entre 2 neurones Plaque motrice = contact entre nerf et muscle
💡 À retenir (TRÈS important)

• Sensibilité consciente : je sens quelque chose → message va vers le cerveau (centripète).
• Mouvement volontaire : je décide de bouger → message part du cerveau vers le muscle (centrifuge).
• Réflexe médullaire : mouvement automatique (ex : retirer la main du feu). Le centre est la moelle épinière, pas le cerveau → c'est très rapide.
• ⚠️ Le cerveau commande le côté OPPOSÉ du corps : l'hémisphère gauche commande le côté droit, et l'inverse.
• ⚠️ La drogue abîme le système nerveux, jamais l'inverse.

Chapitre 2 — Le Système Musculaire

Les muscles, leurs propriétés, comment ils bougent

📖Résumé simple▼
😊 C'est quoi ?

Les muscles font bouger le corps. Quand le nerf donne l'ordre, le muscle se contracte (se raccourcit) et tire sur l'os.

💡 À retenir : les 4 propriétés du muscle

• Excitabilité = le muscle répond quand on le stimule
• Contractilité = le muscle se raccourcit
• Élasticité = le muscle revient à sa taille normale
• Tonicité = le muscle reste un peu tendu, même au repos

🔑 La structure du muscle
Fibre musculaire = cellule allongée, plusieurs noyaux Faisceau = paquet de fibres Tendon = fixe le muscle à l'os Plaque motrice = où le nerf touche le muscle

Chapitre 3 — Les Microbes & l'Immunité ⭐

Le chapitre le plus important de l'examen

📖Résumé simple▼
😊 C'est quoi ?

Les microbes (bactéries, virus) veulent entrer dans le corps et nous rendre malades. Le système immunitaire est l'armée du corps qui les combat.

🔑 Les mots importants
Antigène = le microbe (l'ennemi) Anticorps = arme contre UN microbe précis Phagocytose = une cellule "mange" le microbe Lymphocyte = soldat de l'immunité Plasmocyte = fabrique les anticorps
💡 À retenir (LE PLUS IMPORTANT)

2 types de défense :
1. Non spécifique (rapide) : la phagocytose. Des cellules mangent les microbes. Même réponse pour tous.
2. Spécifique (précise) :
  • Humorale : lymphocytes B → plasmocytes → anticorps
  • Cellulaire : lymphocytes T8 tuent les cellules infectées
  • Lymphocytes T4 = les chefs qui commandent les autres

Vaccination = entraîner le corps → il garde la mémoire → protection longue.
Sérothérapie = donner des anticorps tout prêts → protection rapide mais courte.
